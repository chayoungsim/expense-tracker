# React + TypeScript + Vite

## React 기반 가계부 개인재무관리
```
가계부 + 차트/통계
- CRUD
- 상태관리
- 통계 계산
- 차트
- 필터링
- TypeScript
- React Query
- Zustand
```

### 기술스팩
- framework : React + Vite
- Language : TypeScript
- Styling	: SCSS Module
- Routing	: React Router
- Server State : React Query
- Client State : Zustand
- Form : React Hook Form
- Validation : Zod
- Chart : Recharts
- Date : date-fns
- API	: JSON Server 또는 MockAPI
- Deploy : Vercel


### 기본 CRUD
- 수입 추가

- 삭제

- 수입/지출 비교
- 최근 소비
